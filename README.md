Site destinado á matéria de Desenvolvimento Web do curso Téc em Informática, turma 2025.1

Regras para os colaboradores:

- Classes e IDS devem ser definidos usando CamelCase.
- Revisar oque foi feito antes de commitar.
- SEMPRE colocar na descrição do commit oque foi feito (para facilitar o entendimento).
- indentação clara.
- usar cores, fontes e formas de acordo com o resto do site (caso deseje mudar consulte todos).
- evitar usar imagens com copyright.
